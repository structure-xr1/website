Routage intelligent

Scoring multi-criteres : quota restant (40%), affinite taskType (25%), fiabilite (20%), cout (15%). Le meilleur provider est selectionne automatiquement.

10 types de taches

Code, analyse, redaction, decision, bulk, search, chat, review, long-context, synthese. Chaque type a ses providers optimaux.

🚀

Zero downtime

Fallback automatique : si un provider echoue, le suivant prend le relais. Tracking d'erreurs, providers instables auto-evites.

🏠

Edge-first

Deploye sur Raspberry Pi (733 MB RAM). Dashboard HDMI temps reel, portail web, monitoring systeme integre.

🔒

Multi-controller

7 controleurs specialises : primary, IT, security, supervisor, business, backup, hardware. Chacun gere son domaine.

Token budget

Suivi en temps reel de la consommation par provider. Alertes quota, rotation automatique, historique 7 jours en base SQLite.
